# 🎵 Spotify Music Analytics Dashboard

## 📌 Project Overview

The Spotify Music Analytics Dashboard is an interactive Power BI report designed to analyze Spotify music data and uncover insights into songs, artists, albums, and popularity trends. The dashboard provides a visually appealing interface that enables users to explore music performance metrics and identify key trends across the dataset.

---

## 🎯 Objectives

- Analyze Spotify song and artist data.
- Identify top-performing artists and tracks.
- Track song popularity trends over time.
- Compare album types and content categories.
- Provide an interactive experience through Power BI visualizations.

---

## 📊 Dashboard Features

### 🔹 Key Performance Indicators (KPIs)
- **789** Distinct Songs
- **342** Artists
- **3.28** Average Song Duration
- **90** Average Popularity Score

### 🔹 Song & Artist Analysis
- List of popular songs and artists.
- Artist ranking based on song count.
- Most-streamed songs by artist.

### 🔹 Album Analysis
- Distribution of songs by Album Type.
- Comparison between Albums and Singles.
- Explicit vs Non-Explicit content breakdown.

### 🔹 Trend Analysis
- Average Popularity by Month.
- Distinct Songs Released by Month.
- Songs Released by Year.

### 🔹 Interactive Dashboard
- Cross-filtering between visuals.
- Dynamic data exploration.
- User-friendly dark theme design.

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Spotify Dataset

---

## 📈 Key Insights

- Album releases contribute more songs than singles.
- Popularity scores fluctuate throughout the year.
- A few artists dominate the overall song count.
- Monthly release patterns reveal periods of higher music activity.
- Explicit and non-explicit content can be compared for audience trends.
